🟢 Basic
1. What is Genetic Algorithm?

👉 An optimization technique inspired by natural selection.

2. What is a Neural Network?

👉 A model used for learning patterns from data.

3. What is hybrid GA–NN?

👉 A combination where GA optimizes NN parameters.

🟡 Intermediate
4. What is fitness function?

👉 It measures how good a solution is (here: error).

5. Why use Mean Squared Error?

👉 It measures prediction accuracy.

6. What is crossover?

👉 Combining two parent solutions.

7. What is mutation?

👉 Random modification to maintain diversity.

🟠 Advanced
8. Why does error decrease over generations?

👉 Because GA improves solutions through selection and evolution.

9. Why does error become constant later?

👉 Because the algorithm converges to an optimal solution.

10. Why not use gradient descent alone?

👉 It may get stuck in local minima.

🔴 Tricky Questions
11. What is premature convergence?

👉 When GA stops improving early due to low diversity.

12. How did you avoid it?

👉 By increasing mutation rate and population size.

13. Why is randomness important?

👉 It helps explore the search space.

14. Can GA guarantee best solution?

👉 No, but it finds near-optimal solutions.

# 🔬 4. Important Concepts

| Concept    | Meaning             |
| ---------- | ------------------- |
| Population | Set of solutions    |
| Individual | One solution        |
| Fitness    | Quality of solution |
| Selection  | Choose best         |
| Crossover  | Combine solutions   |
| Mutation   | Random change       |

---

# ⚙️ 5. Working of Genetic Algorithm

### Step-by-step:

1. Initialize population
2. Evaluate fitness
3. Select best individuals
4. Apply crossover
5. Apply mutation
6. Create new generation
7. Repeat until best solution

# 💻 7. Code (From Practical)

```python
import numpy as np
from genetic_algorithm import GeneticAlgorithm
from sklearn.neural_network import MLPRegressor
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error

# Sample data
X = np.random.rand(100, 5)
y = np.random.rand(100)

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Neural Network
nn_model = MLPRegressor(hidden_layer_sizes=(100, 50))

# Fitness function
def fitness_function(params):
    population_size, crossover_rate, mutation_rate = params
    
    ga = GeneticAlgorithm(population_size=population_size,
                          crossover_rate=crossover_rate,
                          mutation_rate=mutation_rate)
    
    nn_model.fit(X_train, y_train)
    
    y_pred = nn_model.predict(X_test)
    mse = mean_squared_error(y_test, y_pred)
    
    return mse

# Parameter ranges
parameter_ranges = {
    'population_size': (50, 100),
    'crossover_rate': (0.6, 0.9),
    'mutation_rate': (0.01, 0.1)
}

# Optimize
ga = GeneticAlgorithm(fitness_function, parameter_ranges)
best_params = ga.optimize()

print("Best Parameters:", best_params)
```
