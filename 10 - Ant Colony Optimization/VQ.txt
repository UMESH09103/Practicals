# 🔷 1. THEORY OF ACO (Easy Explanation)

### 👉 What is ACO?

* ACO = **Ant Colony Optimization**
* It is inspired by **real ants behavior**
* Used to solve **optimization problems** like:

  * Travelling Salesman Problem (TSP)
  * Shortest path finding

---

## 🐜 Real-Life Idea

* Ants move randomly to find food
* While moving, they leave **pheromone (chemical trail)**
* Other ants follow paths with **strong pheromone**
* Shortest path gets more pheromone quickly → becomes best path

👉 So:

* **More pheromone = better path**
* **Short path = more attractive**

---

## 🔁 ACO Working Steps

1. Initialize pheromone on all paths
2. Each ant builds a route
3. Probability depends on:

   * Pheromone (τ)
   * Distance (η)
4. Calculate route length
5. Update pheromone:

   * Evaporation (reduce old pheromone)
   * Add new pheromone (good paths get more)
6. Repeat for many iterations
7. Best path is selected

---

## 📌 Formula (Important for Viva)

Probability of choosing next city:

[
P_{ij} = \frac{(\tau_{ij})^\alpha \cdot (1/d_{ij})^\beta}{\sum (\tau^\alpha \cdot (1/d)^\beta)}
]

Where:

* τ = pheromone
* d = distance
* α = pheromone importance
* β = distance importance
