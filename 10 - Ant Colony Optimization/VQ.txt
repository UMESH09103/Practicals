# 🧠 3. Concept Used

## 🔹 What is MapReduce?

MapReduce = **Distributed data processing model**

### It has 2 main phases:

1. **Map → Process data**
2. **Reduce → Combine results**

---

# ⚙️ 4. Working of Practical 10

### ✔ Step-by-step flow:

1. Input weather dataset
2. Extract **year and temperature**
3. Mapper emits → (year, temperature)
4. Shuffle groups same years
5. Reducer finds:

   * Max temp → hottest year
   * Min temp → coolest year

# 💻 7. Python Code (Simple Version)

```python
# Mapper Function
def mapper(data):
    result = []
    for line in data:
        year, temp = line.split(',')
        result.append((year, int(temp)))
    return result


# Reducer Function
def reducer(mapped_data):
    temp_dict = {}

    # Grouping
    for year, temp in mapped_data:
        if year not in temp_dict:
            temp_dict[year] = []
        temp_dict[year].append(temp)

    # Find min & max
    result = {}
    for year, temps in temp_dict.items():
        result[year] = {
            "max": max(temps),
            "min": min(temps)
        }

    return result


# Main Program
data = [
    "2010,35",
    "2010,38",
    "2011,30",
    "2011,42",
    "2012,25"
]

mapped = mapper(data)
reduced = reducer(mapped)

# Find hottest & coolest year
hottest_year = max(reduced, key=lambda x: reduced[x]["max"])
coolest_year = min(reduced, key=lambda x: reduced[x]["min"])

print("Hottest Year:", hottest_year, reduced[hottest_year])
print("Coolest Year:", coolest_year, reduced[coolest_year])
```
