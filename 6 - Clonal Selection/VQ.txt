## 🔄 **4. Working (Step-by-Step)**

### 1️⃣ Initialize Population

👉 Random solutions create karo

### 2️⃣ Evaluate Affinity

👉 Har solution ki quality check karo

📌 Better solution = high affinity

---

### 3️⃣ Select Best Antibodies

👉 Top solutions choose karo

---

### 4️⃣ Clone (Copy)

👉 Best solutions ko multiple copies me convert karo

### 5️⃣ Mutation

👉 New solutions ki affinity check karo

### 8️⃣ Repeat

## 💻 **6. Code Logic (Simple Explanation)**

👉 Typical steps in code:

---

### 🔹 Random Population

```python id="1a"
population = [random.random() for _ in range(10)]
```

👉 Random solutions generate

---

### 🔹 Fitness Function

```python id="2a"
fitness = -x**2
```

👉 Better value = higher fitness

---

### 🔹 Selection

```python id="3a"
selected = sorted(population, key=fitness)[:best]
```

👉 Best choose karte hain

---

### 🔹 Cloning

```python id="4a"
clones = [item for item in selected for _ in range(n)]
```

👉 Multiple copies

---

### 🔹 Mutation

```python id="5a"
mutated = clone + random.uniform(-0.1, 0.1)
```

👉 Slight change
